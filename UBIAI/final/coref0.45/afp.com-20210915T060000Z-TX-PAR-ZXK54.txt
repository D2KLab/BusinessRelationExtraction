Press Release from ORG: ORG.Yet another demand is met for the construction industry, with precise and efficient excavation work. ORG now has a solution ready that makes it much easier to carry out grading using a loader. Machine operators can now level and setsurfaces precisely using the loader via drawings and GPS, without any assistance from a manual worker and a surveyor. The ORG 3D machine control system is now available for excavators and wheel loaders.Press Release from ORG features multimedia. View Press Release from ORG here: https://www.businesswire.com/news/home/20210914006043/en/The machine industry wants even more 3D control, so ORG now has a solution for wheel loaders (Photo: ORG)The shortage of skilled labour in the construction industry is a growing problem. Contractors are constantly looking for machine operators. ORG meets this challenge using technology that ensures better utilisation of resources on construction sites, by making the individual machine operator more independent and productive.ORG's new 3D solution can be installed on all types of wheel loaders. ORG is the first company in the world that now supports articulated body sensors and installs antennas on the roof.Better machine utilisation3D machine control for loaders was developed in collaboration with ORG's distributors."The construction industry wants to use wheel loaders for even more tasks than moving equipment and loading lorries. Using 3D machine control, the contractor can also use a loader for several tasks, including when surfaces need to be levelled and set. And if the grading has meant that a manhole cover has become hidden, ORG's solution can quickly localise it without having to call in a surveyor. Overall, ORG's technology increases machine utilisation and productivity on the site,"stated PERSON, partner at distributor ORG.With ORG3D fitted on the machine, fast-tracking is ensured, so the system can always follow the machine, and the machine operator can carry out the work precisely."There is a clear need for a simple 3D machine control system that has a short training time and is sold at an affordable price," said PERSON, CCO at ORG.At construction company ORG in Denmark, Technical Manager PERSON is very satisfied with the new technological possibilities: "ORG3D is an excellent tool for ensuring precise grading, no matter the level of experience the machine operator has.
